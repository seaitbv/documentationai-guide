## Overview

`ParticleLibrary` is a centralized library for creating and configuring all particle effects in SpaceFlapper. It provides factory methods for consistent styling, performance optimization through shared textures, and a unified color palette. All particle effects in the game are created through this library.

## Shared textures

`ParticleLibrary` provides cached texture generation methods to avoid redundant rendering:

### Soft particle texture

```swift ParticleLibrary.swift
static func softParticleTexture() -> SKTexture {
    // Generates a 24x24 circle with radial gradient
    // White center fading to transparent edges
    // Used by: rocket boost flames, time warp swirl, trail effects
}
```

The soft particle uses a radial gradient from white (center) to transparent (edges), creating a smooth glow effect when combined with `.add` blend mode.

## Blend modes

All particle effects use `.add` (additive) blend mode for glowing visuals against the dark space background. This creates bright, luminous effects without obscuring the game elements behind them.

<Callout kind="info">
  Additive blending means particle colors add to the background rather than replacing it. Overlapping particles become brighter, creating natural light accumulation. This works particularly well against SpaceFlapper's near-black background.
</Callout>

## Performance considerations

- **Texture caching**: `softParticleTexture()` generates the texture once and reuses it across all emitters
- **Particle limits**: `numParticlesToEmit` caps burst effects to prevent runaway particle counts
- **Auto-removal**: All burst emitters include a timed `SKAction.removeFromParent()` sequence
- **Target nodes**: Trail emitters use `targetNode = scene` so particles stay in world space when the player moves
- **Lifecycle management**: Continuous emitters (trails, flames) set `particleBirthRate = 0` before removal to allow existing particles to fade naturally

## Creating custom effects

All particle effects follow a consistent pattern:

```swift
// 1. Create emitter
let emitter = SKEmitterNode()

// 2. Set texture (use shared when possible)
emitter.particleTexture = ParticleLibrary.softParticleTexture()

// 3. Configure lifecycle
emitter.particleBirthRate = 100
emitter.numParticlesToEmit = 20   // 0 = continuous
emitter.particleLifetime = 0.5
emitter.particleLifetimeRange = 0.2

// 4. Configure movement
emitter.particleSpeed = 80
emitter.emissionAngleRange = .pi * 2  // Full circle

// 5. Configure appearance
emitter.particleColor = ParticleColors.cyan
emitter.particleColorBlendFactor = 1.0
emitter.particleBlendMode = .add

// 6. Configure fade
emitter.particleAlpha = 1.0
emitter.particleAlphaSpeed = -2.0

// 7. Auto-remove for bursts
emitter.run(SKAction.sequence([
    SKAction.wait(forDuration: 1.0),
    SKAction.removeFromParent()
]))
```
